To run the package

1) Open ros NOETIC
2) Go to the catkin_ws directory
3) launch the launch file called cubic_traj_gen.launch 
4) use: roslaunch ar_week8_test cubic_traj_gen.launch
5) This should start the 4 nodes required and open the rqt_plot window
6) In another terminal open the rqt_graph graph 
7) use: rqt_graph
